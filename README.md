# Sistema de Gestión de Tutorías Académicas UNAL

Proyecto para la asignatura **Estructuras de Datos**  
Universidad Nacional de Colombia – Sede Bogotá  

## Descripción del proyecto

El proyecto propone un sistema de gestión de tutorías académicas que reemplace el esquema actual basado en Formularios de Google.  
El sistema tiene como propósito:

- Automatizar la solicitud de tutorías académicas.
- Administrar los tutores disponibles y organizarlos por asignatura.
- Manejar de forma sencilla el histórico de tutorías realizadas.
- Asignar horario y tutor siguiendo un orden de prioridad según la necesidad de los estudiantes.
